# 项目概述

## 一款基于Python的Alist服务管理系统，提供以下功能：

✔️ 可视化服务管理界面  
✔️ 自动IP地址检测与格式化  
✔️ 跨平台配置文件管理  
✔️ 多网络协议支持 (IPv4/IPv6)

# 使用说明        
## 首次配置         
点击 📁 选择程序 按钮定位alist.exe         
在端口输入框设置服务端口（默认5244）         
点击 💾 保存设置         
## 常规操作         
▶️ 启动服务: 主界面 启动服务 按钮         
🔄 重启服务: 修改配置后自动触发         
📋 复制地址: 点击IP地址旁的复制按钮         
⚙️ 重置配置: 通过菜单栏或 重置配置 按钮
